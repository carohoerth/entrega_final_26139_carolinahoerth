# CRUD de Artículos - Backend API

API REST desarrollada con Java y Spring Boot para la gestión y persistencia de artículos, estructurada bajo una arquitectura en capas y en el marco de la POO.
Este proyecto sirve como demostración práctica de la integración de un backend con un cliente (Postman/Navegador) y una base de datos relacional (SQL)

## 🎥 Demostración y Explicación Técnica
Se puede ver el funcionamiento del sistema en vivo y la explicación detallada de su arquitectura en el siguiente enlace: https://www.youtube.com/watch?v=nscwLB7Kgvs

## 🛠️ Tecnologías y Herramientas Utilizadas

- **Lenguaje:** Java 17 / 21
- **Framework Principal:** Spring Boot (Spring Web, Spring Data JPA)
- **Servidor Web:** Apache Tomcat (Embebido en Spring Boot)
- **Base de Datos:** MySQL (Gestionado a través de XAMPP)
- **Persistencia y ORM:** Hibernate / JPA
- **Control de Versiones:** Git
- **Entorno de Desarrollo:** Linux Mint / Visual Studio Code
